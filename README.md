# SmartLot

## Overview
SmartLot is a modern, high‑performance front‑end application built with **React 19** and **Vite 8**. It provides an interactive UI for managing parking lots, leveraging **GSAP** for smooth animations and **Axios** for backend communication.

## Tech Stack
- **Framework**: React 19 (functional components, hooks)
- **Build Tool**: Vite 8 (fast HMR, optimized builds)
- **Styling**: Vanilla CSS with a modular structure (`src/App.css`, `src/index.css`)
- **Animation**: GSAP & `@gsap/react`
- **HTTP Client**: Axios
- **Routing**: React Router DOM
- **Icons**: Lucide‑React, React‑Icons
- **Environment**: dotenv for configuration

## Features
- Responsive layout with dynamic routing
- Animated UI elements using GSAP
- Centralized state handling via React hooks
- Clean, ESLint‑enforced codebase
- Ready‑to‑extend component architecture (`src/componentes/`, `src/vistas/`)

## Getting Started
```bash
# Clone the repository
git clone <repo-url>
cd SmartLot

# Install dependencies
npm install

# Run the development server
npm run dev
```
Open `http://localhost:5173` in your browser.

## Build & Preview
```bash
# Create a production build
npm run build

# Serve the built app locally
npm run preview
```

## Project Structure
```
SmartLot/
├─ .codegraph/            # Codegraph SQLite DB with static analysis metadata
├─ public/                # Static assets
├─ src/                   # Source code

│   ├─ vistas/            # Page‑level view components

└─ vite.config.js         # Vite configuration
```

## Codegraph Insight
The `.codegraph` folder contains `codegraph.db`, a SQLite database generated by the Codegraph tool. It stores a full symbol and call‑site map of the project, useful for static analysis, dependency visualization, and advanced IDE features.
