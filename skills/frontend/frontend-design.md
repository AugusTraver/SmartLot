# Frontend Design Skill

This skill guides creation of distinctive, production-grade frontend interfaces that avoid generic "AI slop" aesthetics. Implement real working code with exceptional attention to aesthetic details and creative choices.

The user provides frontend requirements: a component, page, application, or interface to build. They may include context about the purpose, audience, or technical constraints.

## 1. Design Thinking

### 1.1 Understand the Context
- Purpose: What problem does this interface solve?
- Audience: Who uses it?
- Technical constraints: framework, performance, accessibility.

### 1.2 Commit to a Bold Aesthetic Direction
Pick an extreme direction and execute it with discipline:
- Brutally minimal
- Maximalist chaos
- Retro-futuristic
- Organic / natural
- Luxury / refined
- Playful / toy-like
- Editorial / magazine
- Brutalist / raw
- Art deco / geometric
- Soft / pastel
- Industrial / utilitarian

### 1.3 Clarify Differentiation
- What makes this UNFORGETTABLE?
- What is the one thing someone will remember?

> CRITICAL: Choose a clear conceptual direction and execute it with precision. Bold maximalism and refined minimalism both work - the key is intentionality, not intensity.

## 2. Implementation Goals

Implement working code (HTML/CSS/JS, React) that is:
- Production-grade and functional
- Visually striking and memorable
- Cohesive with a clear aesthetic point of view
- Meticulously refined in every detail

## 3. Frontend Aesthetics Guidelines

### 3.1 Typography
- Choose fonts that are beautiful, unique, and interesting.
- Avoid generic fonts like Arial, Roboto, Inter, or system defaults.
- Opt for distinctive, characterful font choices.
- Pair a striking display font with a refined body font.

### 3.2 Color & Theme
- Commit to a cohesive aesthetic.
- Use CSS variables for consistency.
- Dominant colors with sharp accents outperform timid palettes.

### 3.3 Motion
- Use animations for effects and micro-interactions.
- Prefer CSS-only solutions for HTML.
- Use motion libraries in React when available.
- Focus on high-impact moments: one well-orchestrated page load with staggered reveals creates more delight than scattered animations.
- Use scroll-triggered and hover states that surprise.

### 3.4 Spatial Composition
- Use unexpected layouts.
- Employ asymmetry, overlap, and diagonal flow.
- Break grids with layered elements.
- Choose generous negative space or controlled density to support the design.

### 3.5 Backgrounds & Visual Details
- Create atmosphere and depth rather than defaulting to solid colors.
- Use textures, gradient meshes, noise, geometric patterns, layered transparencies, dramatic shadows, decorative borders, custom cursors, and grain overlays.

### 3.6 Avoid Generic Aesthetics
NEVER use generic AI-generated aesthetics such as:
- Overused fonts: Inter, Roboto, Arial, system fonts
- Clichéd color schemes, especially purple gradients on white backgrounds
- Predictable layouts and component patterns
- Cookie-cutter designs lacking context-specific character

## 4. Creative Interpretation

- Interpret creatively and make unexpected choices that feel genuinely designed for the context.
- No design should be the same.
- Vary between light and dark themes, font choices, and aesthetic directions.

> NEVER converge on common choices (Space Grotesk, for example) across generations.

## 5. Match Complexity to Vision

- Maximalist designs need elaborate code with extensive animations and effects.
- Minimalist or refined designs need restraint, precision, and careful attention to spacing, typography, and subtle details.
- Elegance comes from executing the vision well.

## 6. Final Reminder
Remember: Claude is capable of extraordinary creative work. Don't hold back; show what can truly be created when thinking outside the box and committing fully to a distinctive vision.
