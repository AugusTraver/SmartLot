# SmartLot Design System

## Direccion Visual
- Estilo SaaS moderno inspirado en Linear y Vercel.
- Minimalista, profesional, oscuro por defecto y con alto nivel de legibilidad.
- Superficies sobrias, bordes sutiles, iconografia simple y espaciado consistente.

## Fundamentos
- Fondo principal: tonos zinc/neutral muy oscuros.
- Superficies: contraste leve entre pagina, paneles y elementos interactivos.
- Bordes: sutiles para separar sin recargar.
- Tipografia: clara, compacta y orientada a datos.
- Radios: moderados, preferentemente `rounded-md` o `rounded-lg`.

## Componentes Base
- Button: variantes primary, secondary, ghost y danger.
- Input: label claro, helper text opcional, error visible.
- Badge: estados neutral, success, warning, danger e info.
- Table: headers sobrios, filas escaneables, acciones alineadas.
- Card: solo para unidades de informacion, no para envolver secciones completas.

## Responsive
- Desktop: dashboards con grillas, tablas amplias y filtros visibles.
- Tablet: columnas reducidas y paneles apilados parcialmente.
- Mobile: navegacion compacta, metricas apiladas y tablas adaptadas a tarjetas o filas resumidas.

## Buenas Practicas
- Usar iconos para acciones frecuentes con tooltip cuando sea necesario.
- Mantener alineacion numerica en metricas, importes y porcentajes.
- Destacar alertas operativas sin saturar la pantalla.
- Reservar colores intensos para estados y acciones importantes.

## Patrones a Evitar
- Paletas dominadas por un solo color.
- Sombras fuertes o efectos decorativos innecesarios.
- Gradientes como base del producto.
- Texto pequeno con bajo contraste en tablas o formularios.
